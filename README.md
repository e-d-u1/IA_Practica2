# Práctica de Sistemas Basados en el Conocimiento - Recomendación de viviendas -  CLIPS

Hay dos versiones de lo mismo, los archivos número_módulo.clp es para tenerlos divididos, y el prototipo3.clp tiene  la división modular en el mismo fichero.

La ontologia de Protégé está en mi-ontologia.ttl y también en input.clp con su versión final traducida a CLIPS.

Se separan tambien las instancias por si se quier comparar o ver facilmente a qué vivienda exactamente corresponde el resultado mostrado


## Cómo ejecutar

La práctica se ha desarrollafo con CLIPS 6.31 y Protégé 5.6.7, se ha probando tanto en entornos linux como windows.

A: Interfaz Gráfica
1. Abrir CLIPS  
2. Ctrl+L: prototipo3.clp (Se deben añadir las instancias después de definir las clases)
3. Ctrl+R,Ctrl+E,Ctrl+R   
Y ya debería funcionar correctamente

B: Consola
1. Cargar simplemente comandosParaEjecutar.txt en la consola o Ctrl+C y Ctrl+V
